# Crime Data Analysis & Power BI Dashboard

End-to-end Business Intelligence project analyzing Indian crime data (2020–2024).  
Built using Power BI, DAX, and Power Query to reveal hotspots, seasonal trends, and case-closure KPIs.

**Files Included**
- `Crimes in india.pbix` – Power BI dashboard file  
- `crime_dataset_india.csv` – Dataset  
- `Crime Data.docx` – Project report
